Glossary of Terms

HTML
HyperText Markup Language, used for structuring web pages.
CSS
Cascading Style Sheets, used to style HTML elements.
JavaScript
A programming language that enables interactivity on webpages.
Element
A single HTML tag with its content, like <p>Hello</p>.
Attribute
Extra information inside a tag, e.g., href in <a href="...">.
Semantic HTML
HTML that describes meaning, like <header> or <article>.
Viewport
The visible area of a web page on a device screen.
Responsive Design
Design approach that adjusts layout to different screen sizes.

Steps to Create a Web Page

  1. Plan your content
    1. Decide on headings
    2. Prepare text and images
  2. Write HTML structure
    1. Use semantic tags
    2. Add content in appropriate sections
  3. Style with CSS
    1. Link external stylesheet
    2. Apply colors, fonts, and layout
  4. Add interactivity with JavaScript
  5. Test on different devices
